### Step 1: Post Arrives as Draft
- Automation sends post to Substack via email-to-post

- Nothing publishes automatically

### Step 2: Review Content

### Step 3: Edit if Needed
- Add context or insights
- Fix formatting issues
- Remove items that don't meet standards
- Enhance summaries if needed

### Step 4: Set Subscription Tier

**For Daily Posts:**
- Leave as **"Free"** (default)
- All subscribers can read

**For Weekly Newsletters:**
- Set as **"Paid subscribers only"**
- Only paid subscribers can read

**How to Set Tier:**
1. Open the draft post
2. Click **"Settings"** (or gear icon)
3. Find **"Subscription tier"** section
4. Select:
   - **"Free"** for daily posts
   - **"Paid subscribers only"** for weekly newsletters
5. Save changes

### Step 5: Publish
- Click **"Publish"** when ready
- Or schedule for a specific time
- Post goes live to your subscribers
